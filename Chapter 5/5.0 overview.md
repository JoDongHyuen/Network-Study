Network layer control plane overview
=====================

* Routing table을 생성하는 Algorithm에 대해서 알아볼거임
  ```
  ▶전통적인 라우팅 알고리즘

  ▶SDN 컨트롤러

  ▶network management, configuration
  ```
* Routing algorithm을 구체적인 Protocol로 지정해놓은 것에 대해 알아볼거임
  ```
  ▶OSFP, BGP, RIP

  ▶SDN과 관련된 OpenFlow, ODL and ONOS 컨트롤러, ICMP

  ▶SMNP, YANG / NETCONF
  ```
